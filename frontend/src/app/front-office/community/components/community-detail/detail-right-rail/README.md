# Detail Right Rail Subcomponent

`DetailRightRailComponent` renders supplemental right-column information and actions.

## Files

- `detail-right-rail.component.ts`: required `host` input.
- `detail-right-rail.component.html`: metadata blocks and secondary actions.
- `detail-right-rail.component.css`: right-rail visual treatment.

## Notes

- This subcomponent should not trigger data fetches directly.
- Keep high-level action handlers routed through host methods.
